# 11.158 Final Project - Analyzing MBTA Effectiveness for MIT Student Access to Greater Boston

## Acquire Datasets

Create a `data` folder containing:
- Download and unzip into `data`:
    - https://www.mbta.com/developers/gtfs
    - https://mbta-massdot.opendata.arcgis.com/datasets/5f71a5c035fc4a4dad1b7fa73ba27ef8
- Download CSV's and place in `data`:
    - https://mbta-massdot.opendata.arcgis.com/maps/e09d680af4a0441eb49ce4ef7b1796eb
    - https://mbta-massdot.opendata.arcgis.com/maps/7acd353c1a734eb8a23caf46a0e66b23
    - https://mbta-massdot.opendata.arcgis.com/maps/80a379ebaa374cfd836ca4d3880ceda4

## Running

Set the constants at the top of `main_v2.py` as desired.

```bash
python main_v2.py
```
